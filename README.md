# متجر العفراء - متجر نسائي فاخر

موقع ويب احترافي وفخم لمتجر نسائي بتصميم عصري وأنيق.

## المميزات الرئيسية

### 1. واجهة المتجر الفخمة
- تصميم فخم وعصري بألوان ناعمة (وردي، بيج، ذهبي)
- عرض المنتجات بشكل بطاقات جميلة
- تأثيرات انتقالية سلسة وأنيقة
- تجربة مستخدم احترافية جداً

### 2. نظام الشات الداخلي
- دردشة مباشرة مع العملاء من داخل الموقع
- تسجيل دخول آمن بـ username و password
- حفظ المحادثات السابقة محلياً
- كل مستخدم له محادثته الخاصة

### 3. لوحة التحكم الإدارية
- إضافة منتجات جديدة
- تعديل وحذف المنتجات
- قراءة رسائل العملاء
- الرد على العملاء مباشرة
- كل العمليات بدون إعادة تحميل الصفحة

### 4. التخزين والبيانات
- **LocalStorage**: لحفظ بيانات المستخدم والمحادثات محلياً
- **Firebase Firestore** (اختياري): لحفظ البيانات في السحابة

## كيفية الاستخدام

### للزوار
1. افتح `index.html` في المتصفح
2. تصفح المنتجات المتاحة
3. أضف المنتجات للسلة
4. استخدم الشات للتواصل مع المتجر

### للمشرف
1. انتقل إلى `admin.html`
2. أضف منتجات جديدة
3. أدر المنتجات الموجودة
4. رد على رسائل العملاء

## التكامل مع Firebase (اختياري)

لتفعيل Firebase:

1. أنشئ مشروع جديد على [Firebase Console](https://console.firebase.google.com)
2. استخرج بيانات المشروع (API Key, Project ID, إلخ)
3. استبدل البيانات في `js/main.js`:

```javascript
const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "your-project.firebaseapp.com",
    projectId: "your-project",
    storageBucket: "your-project.appspot.com",
    messagingSenderId: "YOUR_SENDER_ID",
    appId: "YOUR_APP_ID"
};
```

4. فعّل Firestore في Firebase Console

## النشر على Netlify

### الطريقة الأولى: Netlify Drop
1. اذهب إلى [Netlify Drop](https://app.netlify.com/drop)
2. اسحب ملفات المشروع (أو ملف ZIP)
3. سيتم نشر الموقع فوراً

### الطريقة الثانية: Git
1. رفع المشروع على GitHub
2. ربط المستودع بـ Netlify
3. سيتم النشر التلقائي
